# 副动词 (Converb / дагавар үйл үг)

## 核心概念

副动词是动词的连接形式，相当于静词类的格的形式。它不能充当独立句的述语，是典型的非结束式。副动词后边连接的只能是动词。

## 分类

副动词按连接特点分为三类：

### 第一类：单纯连接副动词

#### 1. 并列副动词 -ж / -ч

连接普通动词与助动词，或连接两个独立动词。

| 功能 | 示例 | 含义 |
|------|------|------|
| 连接助动词 | яв**ж** байна | 正在走 |
| 连接动词 | ид**ж** ууна | 吃（并且）喝 |
| 重复表示体 | идэж идээд | 吃了又吃 |

#### 2. 先行副动词 -аад / -ээд / -оод / -өөд

表示前一个行为完成后，后一个行为开始（完成体）。

| 示例 | 含义 |
|------|------|
| ав**аад** явна | 拿了走 |
| ир**ээд** хэлжээ | 来之后说的 |
| хэл**ээд** иржээ | 说了之后来的 |

#### 3. 共同副动词 -н

主要用于书面语，表示重复同一动作或"立刻"的意义。

| 示例 | 含义 |
|------|------|
| нис**эн** одов | 飞去了 |
| бүтээн байгуулах | 建设 |

### 第二类：制约连接副动词

这类副动词表示后边主要动词的条件，充当动词性状语。

#### 4. 立刻副动词 -магц / -мэгц

前一个动作发生后很快发生后一个动作。

| 示例 | 含义 |
|------|------|
| ир**мэгц** хэлсэн | 一来就说了 |
| хонх дуугар**магц** хичээл эхлэв | 一打铃就上课了 |

#### 5. 跟随副动词 -хлаар / -хлээр

在前一个动作发生的时间和条件下发生后一个动作。

| 示例 | 含义 |
|------|------|
| чамайг хэлэх**лээр** сая мэджээ | 你说，我才知道 |
| чанаад идэх**лээр** их сайн | 煮着吃很好吃 |

#### 6. 前提副动词 -манжин / -мэнжин

前一个动作是后一个动作的前提。后边常有 **сая** (才) 呼应。

| 示例 | 含义 |
|------|------|
| сур**манжин** (сая) чадна | 学习之后才会 |
| чи хэлмэнжин би сая мэдлээ | 你说了，我才知道 |

#### 7. 假定副动词 -вал / -вэл

表示前一个动作是假设的动作或前提条件。否定用 **-хгүй бол**。

| 示例 | 含义 |
|------|------|
| ав**вал** өгнө | 如果要就给 |
| бай**вал** аваад ир | 有的话就拿来 |
| маргааш бороо ор**вол** явлагагүй | 明天下雨的话就不要走了 |

#### 8. 让步副动词 -вч

表示让步和即使。否定句中表示即使前一个动作实现了，后一个动作也不能实现。

| 示例 | 含义 |
|------|------|
| өгө**вч** авахгүй | 给也不要 |
| хэн хэлэ**вч** би явахгүй | 谁说我也不去 |
| тэр явлаа гээд горьгүй юм ээ | 他去了也是无用 |

#### 9. 界限副动词 -тал / -тэл

表示动作达到的限度，或后一个动作继续到前一动作的实现。

| 示例 | 含义 |
|------|------|
| ир**тэл** хүлээе | 等到回来 |
| нар гар**тал** унтжээ | 睡到太阳出来了 |

#### 10. 目的副动词 -хаар / -хээр

表示前一个动作是后一个动作的目的。

| 示例 | 含义 |
|------|------|
| бичиг ава**хаар** иржээ | 取书来了 |
| ус ава**хаар** явжээ | 取水去了 |

#### 11. 趁机副动词 -нгаа / -нгээ

表示顺便或"临……时候"做某动作。

| 示例 | 含义 |
|------|------|
| ява**нгаа** дайраад гаръя | 临走的时候路过一下 |
| ирэ**нгээ** аваад ир | 来的时候拿来吧 |

### 第三类：混合连接副动词

#### 12. 延续副动词 -саар / -сээр

既有单纯连接特点，又有制约连接特点，表示延续。

| 功能 | 示例 | 含义 |
|------|------|------|
| 连接助动词 | тэр хүн одоо болтол хүлээ**сээр байна** | 那个人一直到现在还在等着 |
| 表示条件 | хээр гадаа гүй**сээр** бүх шөнийг мэдсэнгүй өнгөрүүл | 在野外奔跑着不知不觉地过了一夜 |

## 副动词总表

| 类别 | 名称 | 后缀 | 功能 |
|------|------|------|------|
| 单纯连接 | 并列 | -ж / -ч | 并列、进行 |
| | 先行 | -аад / -ээд | 完成体 |
| | 共同 | -н | 书面语、立刻 |
| 制约连接 | 立刻 | -магц / -мэгц | 马上 |
| | 跟随 | -хлаар / -хлээр | 跟随 |
| | 前提 | -манжин / -мэнжин | 前提 |
| | 假定 | -вал / -вэл | 条件 |
| | 让步 | -вч | 即使 |
| | 界限 | -тал / -тэл | 直到 |
| | 目的 | -хаар / -хээр | 目的 |
| | 趁机 | -нгаа / -нгээ | 顺便 |
| 混合连接 | 延续 | -саар / -сээр | 持续 |

## 派生连词

部分副动词可派生为连词：

| 副动词 | 派生词 | 含义 |
|--------|--------|------|
| гэх (说) | гэтэл | 但是 |
| хүрэх (到达) | хүртэл | 直到 |

## 例句

> Би түүнийг ирнэ гэж бодсон. **Гэтэл** ирээгүй.
> (我以为他会来。但是（他）没来。)
