# 使役式 (Causative / хүчиндүүлэх)

## 核心概念

使役式表示"使/让某人做某事"，通过在动词词干后加使役后缀构成。

## 详细解释

### 常见使役后缀

| 原动词 | 使役形式 | 含义 |
|--------|---------|------|
| сурах (学) | сургах (教) | learn → teach |
| бичих (写) | бичүүлэх (让写) | write → have sb. write |
| идэх (吃) | идүүлэх (喂) | eat → feed |
| үзэх (看) | үзүүлэх (让看) | see → show |
| хийх (做) | хийлгэх (让做) | do → have sb. do |

### 后缀变体

使役后缀有多种形式：
- **-га- / -гэ- / -го- / -гө-**
- **-лга- / -лгэ-**
- **-уул- / -үүл-**

## 用法

使役式用于：
- 表达命令、请求（间接）
- 表达"让/使/叫"某人做某事

## 例句

> Би нохойдоо мах идүүлсэн.
> (我让狗吃了肉。)
