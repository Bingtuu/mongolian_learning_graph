# SOV 语序 (Subject-Object-Verb / өгүүлбэрийн бүтэц)

## 核心概念

蒙古语的基本语序是 SOV（主语-宾语-谓语），即动词通常位于句子的末尾。

## 详细解释

### 基本语序

```
主语 + 宾语 + 谓语
Би    + ном  + уншина.
我     书     读
"我读书。"
```

### 语序灵活性

虽然 SOV 是基本语序，但在实际使用中，蒙古语的语序相对灵活，语序变化通常用于强调。

## 关联知识

- [[名词]] — 主语和宾语通常由名词或代词充当
- [[动词]] — 谓语由动词充当
