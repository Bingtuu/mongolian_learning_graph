# 动词词干的层次 (Verb Stem Hierarchy / үйл үгийн язгуур)

## 核心概念

蒙古语动词词干就是去掉式动词、形动词和副动词的各种后缀之后剩下的部分，**形式上与式动词第二人称的命令形式相同**。形动词的后缀从其动词性变化来说是结尾后缀，从其静词性变化来说是一种词干后缀。

理解动词词干的层次，是拆解蒙古语长动词、掌握其构词与构形规律的关键。

> 来源: 清格尔泰语法 2.3.1

---

## 词干层次总览

清格尔泰将蒙古语动词词干分为**四个层次**：

```
词根 → 一次词干 → 二次词干 → 态性词干 → 体性词干 → 式/形动词/副动词词尾
```

| 层次 | 构成方式 | 后缀性质 | 示例 |
|------|----------|----------|------|
| **词根** | 最原始形式 | — | бол (熟), үйл (事/活儿) |
| **一次词干** | 词根 + 派生后缀 | 构词后缀 | боловс (成熟), үйлд (做) |
| **二次词干** | 一次词干 + 体性派生后缀 | 构词为主 | анивхий (眨眼), татгалза (犹豫) |
| **态性词干** | 一次/二次词干 + 态后缀 | 构形为主，兼有构词 | явуул (让走), барилд (摔跤) |
| **体性词干** | 态性词干 + 体后缀 | 纯构形后缀 | явуулчих (派遣了), барилдчих (摔跤了) |

---

## 第一层：一次词干

一次词干包括所有**非派生词干**和由其它词类（名词、形容词、数词、时位词、副词、摹拟词等）**派生的动词词干**。

### 由静词类派生的一次词干

| 后缀 | 说明 | 示例 |
|------|------|------|
| **-лах/-лэх** | 与该事物有关的动作 | хайч (剪刀) → хайч**ла** (剪) |
| **-дах/-дэх** | 与该事物有关的动作 | чулуу (石头) → чулуу**да** (抛石头) |
| **-дах/-дэх** | 产生相应特征 | илүү (多余) → илүү**дэ** (超过) |
| **-дах/-дэх** | 与该事物有关 | гал (火) → гал**да** (放火) |
| **-жих** | 变成/具备该特征 | залуу (年轻) → залуу**жи** (变年轻) |
| **-ших** | 形成或具备 | зун (夏) → зун**ши** (入夏) |
| **-рах/-рэх** | 具备该特征 | цагаан (白) → цагаа**ра** (变蓝/熟练) |
| **-шаах/-шээх** | 看作具有该性质 | сайн (好) → сай**шаа** (赞扬) |
| **-рхах/-рхэх** | 以该特征为自负 | сайн (好) → сай**рха** (自夸) |
| **-члах/-члэх** | 使之成为 | боол (奴隶) → боол**чло** (奴役) |
| **-цах/-цэх** | 接受该特征 | халуун (热) → халуу**ца** (感到热) |
| **-сах/-сэх** | 渴望该事物 | мах (肉) → мах**са** (爱吃肉) |

### 由摹拟词派生的一次词干

| 后缀 | 说明 | 示例 |
|------|------|------|
| **-гих** | 持续的声音 | шуу (响声) → шуу**ги** (喧哗) |
| **-глах/-глэх** | 放声发出 | вааг (驴叫声) → вааг**ала** (驴叫) |
| **-гинах/-гинэх** | 乐音 | дан (叮当) → дан**гина** (叮当作响) |
| **-храх/-хрэх** | 强大声音 | хүр (轰响) → хүр**хрэ** (咆哮) |

### 由造形副词派生的一次词干

| 后缀 | 说明 | 示例 |
|------|------|------|
| **-чих** | 断然作出该结果 | хага (碎裂地) → хага**чи** (完全打破) |
| **-лах/-лэх** | 作出该结果的及物动词 | тас (断) → тас**ла** (扯断) |
| **-рах/-рэх** | 该结果的不及物动词 | тас (断) → тас**ра** (脱离) |

> **及物与不及物的区分**：派生动词的及物与不及物差别可用其后缀区分。`-лах/-лэх` 常为及物，`-рах/-рэх` 常为不及物。

## 第二层：二次词干

二次词干是在**一次词干**上附加体性派生后缀而构成的。这类后缀数量不多，使用范围有限，往往只用于词汇意义比较独特的一部分词。

### 构成二次词干的后缀

| 后缀 | 示例 | 说明 |
|------|------|------|
| **-вхий/-вхийлэ** | ань (合眼) → **анивхий** (眨眼) | 瞬时动作 |
| **-вчи** | ань → **анивчи** (不断眨眼) | 反复动作 |
| **-ла** | нух (揉) → **нухла** (不住地揉搓) | 多次体 |
| **-ни** | тогто (定) → **тогтни** (安定) | 状态化 |
| **-мда** | найра (和好) → **найрмда** (和解) | 建立关系 |
| **-лза** | найга (摇晃) → **найгалза** (袅袅飘动) | 反复轻微动作 |
| **-валз/-мэлзэ** | тата (拉) → **татвалза** (痉挛) | 不规则颤动 |
| **-галза** | хара (看) → **харгалза** (关怀) | 趋向性动作 |
| **-гана** | сарва (伸手) → **сарвагана** (挣扎) | 频繁小动作 |

> **特点**：二次词干后缀**构词成分大于构形成分**。它们派生的词干大都有自己独特的词汇意义，如 `тата` (拉) 与 `татгалзах` (犹豫) 的词汇意义显然不同。

---

## 第三层：态性词干

态性词干是在一次词干或二次词干上附加**态的后缀**而构成的。态的后缀主要表示行为与其主体之间的关系。

### 态后缀的分类

| 态 | 后缀 | 示例 | 说明 |
|----|------|------|------|
| **自动态** | 无专门后缀 | ном бичиг үзнэ (看书) | 主体自己完成 |
| **使动态** | -уул/-үүл, -лга/-лгэ, -га/-гэ, -аа/-ээ | яв**уул** (让走), суул**га** (让坐) | 令他人进行 |
| **被动态** | -гд | бари**гда** (被捕), цохь**гдо** (被打) | 主体遭受动作 |
| **互动态** | -лд | барил**д** (摔跤), алал**д** (撕杀) | 互相动作 |
| **共动态** | -лц | барил**ц** (同捕捉), туслал**ц** (同帮助) | 一起动作 |
| **众动态** | -цгаа/-цгээ | яв**цгаа** (大家走), суул**цгаа** (大家坐) | 众人共同动作 |

### 态后缀的性质

态的后缀**既具有词汇意义，又具有语法意义**：

1. **语法意义大**：表示动词的态；
2. **不受词汇意义限制**：能普遍使用；
3. **能派生新词**：如 `явуулалга` (行为), `тулалдаан` (战役), `солилцоо` (交易)。

> 从总体看，态后缀的**语法意义比较突出**，因此可看作**构形后缀**。

### 态后缀的重叠使用

不同态的后缀可以**重叠**接加：

> танил**ц** + **уу** + **ла** = танилцуулах (使之认识/介绍)
> 
> танилцуулах + **лц** + **гаа** = танилцуулацгаах (大家使之认识)
> 
> үз**эгд** + **үү** + **лэ** = үзэгдүүлэх (使之被看见)
> — 来源: [[蒙古语语法（西里尔）— 清格尔泰]] | 语法点: [[动词词干的层次]]

## 第四层：体性词干

体性词干是在一次词干、二次词干或态性词干上附加**体的后缀**而构成的。体后缀主要给动词补充语法意义。

### 体的后缀

| 体 | 后缀 | 示例 | 说明 |
|----|------|------|------|
| **暂短体** | -схийх | бай**схийх** (待一会儿), амс**хийх** (休息一会儿) | 行为短暂 |
| **完成体** | -чих | яв**чих** (走了), ид**чих** (吃了) | 行为已完成 |
| **请求完成体** | -аадах/-ээдэх | яваа**дах** (请走吧), хэлээ**дэх** (请说吧) | 希望完成，带亲切色彩 |

### 体后缀的性质

体的后缀是**纯构形后缀**：

1. **语法意义大**：表达动词的体；
2. **普遍适用**：能广泛使用（`-схийх` 范围较小）；
3. **不能再派生新词**：区别于前面所有后缀；
4. **位置固定**：是动词词干的最后部分，前面可接态后缀，后面接式/形动词/副动词词尾。

### 体后缀 + 式词尾的实例

> яв**уул** + **чих** = явуулчих (派遣了)
> 
> барил**д** + **чих** = барилдчих (摔跤了)
> 
> яв**цгаа** + **чих** = явцгаачих (大家走了)
> 
> хэлэл**ц** + **чих** = хэлэлцэчих (讨论了)
> 
> явуул**аадах** = явуулаадах (请派遣)
> 
> барилд**аадах** = барилдаадах (请摔跤)
> — 来源: [[蒙古语语法（西里尔）— 清格尔泰]] | 语法点: [[动词词干的层次]]

## 完整拆解示例

### боловсруулах (训练)

| 层次 | 形式 | 说明 |
|------|------|------|
| 词根 | **бол** | 熟 |
| 一次词干 | бол + васан → **боловс** | 成熟的 |
| 二次词干 | боловс + рох → **боловсро** | 成熟/熟练 |
| 态性词干 | боловсро + уул → **боловсруул** | 使成熟/训练 |
| 形动词 | боловсруул + ах → **боловсруулах** | 训练的（形动词）|

### үйлдвэрлэлийн (生产的)

| 层次 | 形式 | 说明 |
|------|------|------|
| 词根 | **үйл** | 事/活儿 |
| 一次词干 | үйл + дэх → **үйлд** | 做 |
| 二次词干 | үйлд + вэр → **үйлдвэр** | 工业/工厂 |
| 一次词干 | үйлдвэр + лэх → **үйлдвэрлэ** | 生产 |
| 名词化 | үйлдвэрлэ + л → **үйлдвэрлэл** | 生产 |
| 属格 | үйлдвэрлэл + ийн → **үйлдвэрлэлийн** | 生产的 |

## 词缀层次的一般规律

蒙古语词缀的接加次序一般是：

```
构词词缀 → 构形词缀 → 结尾词缀
```

- **构词词缀**：构成一次词干、二次词干（-лах, -дах, -вхий, -лза 等）
- **构形词缀**：态后缀（-уул, -лд, -гд, -лц, -цгаа）
- **结尾词缀**：体后缀（-чих, -схийх, -аадах）→ 式/形动词/副动词词尾

> 在少数情况下，构词后缀可在某些构形后缀（态后缀）之后接加。在极个别情况下，结尾词缀之后还出现构词构形后缀，如 `танай**хан**` (你们的人)。
> — 来源: [[蒙古语语法（西里尔）— 清格尔泰]] | 语法点: [[动词词干的层次]]
