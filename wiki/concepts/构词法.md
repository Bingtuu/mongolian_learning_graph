# 构词法 (Word Formation / үг бүтэх зүй)

## 概述

构词法研究的是如何在原有词汇基础上产生新词的方法。蒙古语主要通过**派生**（加词缀）和**复合**（词的组合）两种方式构词。

## 构词法与构形法的区别

| | 构形法 | 构词法 |
|--|--------|--------|
| 研究对象 | 一个词的语法变化形式 | 产生新词的方法 |
| 词汇意义 | 不改变 | 改变 |
| 普遍性 | 普遍适用（如格的变化） | 局限性和独特性 |

> малын (牲畜的) → **构形**（属格）
> мал → малчин (牧民) → **构词**（新词）
> — 来源: [[蒙古语语法（西里尔）— 清格尔泰]] | 语法点: [[构词法]]

## 蒙古语词的结构

蒙古语词一般可分为：**词根 + 词缀**

- **词根**：由一个或两个音节构成
  - 独立词根：гар (手), хүн (人) — 可单独使用
  - 非独立词根：ца (纸/雪/白的词根) — 不能单独使用

- **词缀**：后缀依次相加
  - 构词词缀 → 构形词缀 → 结尾词缀

### 词缀层次示例

**боловсруулах** (训练):
- бол (词根：熟)
- → боловс (加 васан：成熟的)
- → боловсро (加 рох：成熟/熟练)
- → боловсруул (加 уул：使成熟/训练)
- → боловсруулах (加 х：形动词)

**үйлдвэрлэлийн** (生产的):
- үйл (词根：事/活儿)
- → үйлд (加 дэх：做)
- → үйлдвэр (加 вэр：工业/工厂)
- → үйлдвэрлэ (加 лэх：生产)
- → үйлдвэрлэл (加 л：生产)
- → үйлдвэрлэлийн (加 ийн：定语)

## 复合词

两个词结合紧密，产生新词汇意义的词组。

### 特点
- 形式固定，不能随意调换或增减
- 有与原义不同的新词汇意义
- 有些语音形式也起变化

### 示例

| 复合词 | 直译 | 实际意义 |
|--------|------|----------|
| төмөр зам | 铁 + 路 | 铁路 |
| галт тэрэг | 火 + 车 | 火车 |
| морьт цэрэг | 骑马 + 兵 | 骑兵 |
| хар архи | 黑 + 酒 | 白酒 |
| цагаан толгой | 白 + 头 | 字母 |
| ямаа хөх | 山羊 + 乳房 | 白薇（植物） |
| үхэрийн нүд | 牛 + 眼睛 | 茅藤子（植物） |
| туулайн хөөр | 兔子 + 腰子 | 栗子 |
| хүйс тэмтрэх | 肚脐 + 摸 | 满门抄斩 |
| босч явах | 起来 + 走 | 逃走 |

## 动词的派生

### 由静词类派生动词

| 后缀 | 含义 | 示例 |
|------|------|------|
| -лах/-лэх | 与该事物有关的动作 | ус → ус**лах** (灌溉) |
| -рах/-рэх | 产生相应特征 | хөх → хөх**рөх** (变蓝) |
| -дах/-дэх | 与该事物有关的动作 | гал → гал**дах** (放火) |
| -рхах/-рхэх | 以该特征为自负 | сайн → сай**рхах** (自夸) |
| -члах/-члэх | 使之成为该事物 | боол → боол**члох** (奴役) |
| -жих | 变成该事物/具备该特征 | залуу → залуу**жих** (变年轻) |
| -ших | 形成或具备该事物 | зун → зун**ших** (入夏) |
| -шаах/-шээх | 看作具有该性质 | сайн → сай**шаах** (赞扬) |
| -шрах/-шрэх | 具备该特征 | зүг → зүг**шрэх** (逐渐适应) |
| -цах/-цэх | 接受该特征 | халуун → халуу**цах** (感到热) |
| -сах/-сэх | 渴望该事物 | мах → мах**сах** (爱吃肉) |

## 句法构词法（复合词）

### 常用构词成分

蒙古语中许多常用词可以构成大量复合词：

| 词 | 词性 | 可构成 |
|----|------|--------|
| ам (口) | 名词 | 复合名词、动词、形容词、副词 |
| гар (手) | 名词 | 复合名词、动词、形容词、副词 |
| орох (进入) | 动词 | 复合动词 |
| алдах (丢失) | 动词 | 复合动词 |
| хар (黑) | 形容词 | 复合名词、动词、形容词、副词 |
| халуун (热) | 形容词 | 复合名词、动词、形容词、副词 |
| нэг (一) | 数词 | 复合名词、动词、形容词、副词、量词 |

### 复合词的内部结构

#### 复合名词

| 结构 | 示例 |
|------|------|
| **并列** | өдөр шөнө (日夜), эр эм (夫妻) |
| **辅助** | зан ааль (性格), юм хум (东西) |
| **定体** | төмөр зам (铁路), гүн ухаан (哲学) |
| **宾述** | тоо бодлого (算术), ус төрөгч (氢) |
| **状述** | зохион байгуулалт (组织) |
| **主述** | усгүйшэл (脱水) |

#### 复合形容词

| 结构 | 示例 |
|------|------|
| **并列** | муруй сарий (弯弯曲曲), бат бөх (坚固) |
| **定体** | гал улаан (火红), дүн цагаан (洁白) |
| **主述** | хөл хүнд (怀孕的), санаа амар (安心的) |

#### 复合动词

| 结构 | 示例 |
|------|------|
| **并列** | ирэх очих (来往), өгөх авах (授受) |
| **宾述** | хүйс тэмтрэх (斩尽杀绝), үг дагах (听话) |
| **主述** | хар дарах (梦魇), толгой эргэх (头昏) |
| **状述** | хага цохих (打破), задлан шинжлэх (分析) |

#### 复合副词

| 结构 | 示例 |
|------|------|
| **并列** | байн байн (再三), энд тэндгүй (到处) |
| **辅助** | арай царай (勉勉强强), санд мэнд (匆匆忙忙) |
| **定体** | цаг зуур (暂时), агшин зуур (瞬间) |

### 复合词的特点

1. **后边的词是中心词**：复合词的词性由后边词的词性决定。
2. **词形变化在后边词**：复合词的语法变化体现在后边词上。
3. **意义不能从字面推断**：如 хүйс тэмтрэх ≠ "摸肚脐"，而是"满门抄斩"。
