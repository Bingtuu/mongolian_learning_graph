# 助词 ч (Particle / туслах үг "ч")

## 规则概述

助词 **ч** 是蒙古语中使用非常广泛的一个助词，主要含义是"甚至""也"，也可表达"无论……"的泛指意义。

---

## 基本用法："甚至""也"

**ч** 直接附加在词后，表示"甚至……也……"：

| 示例 | 含义 |
|------|------|
| Хүүхэд **ч** мэднэ. | （甚至）小孩子也懂。 |
| Надад нэг **ч** мөнгө байхгүй. | 我这里一个子儿都没有。 |
| Би **ч** бас явна. | 我也走。 |

---

## 与疑问词联用："无论……"

**ч** 与疑问词结合，表示"无论……""不管……"：

| 形式 | 含义 |
|------|------|
| ямар **ч** | 无论哪样、无论如何 |
| юу **ч** | 无论什么 |
| хэн **ч** | 无论谁 |
| хэзээ **ч** | 无论何时 |
| аль **ч** | 无论哪个 |

---

## 典型例句

| 示例 | 含义 |
|------|------|
| Эднээс аль нь болох вэ? | 这里头哪个行？ |
| Аль нь **ч** болно. | 哪个都行。 |
| | |
| Би хэзээ явах вэ? | 我该什么时候去？ |
| Хэзээ **ч** болно. | 什么时候都可以。 |
| | |
| Чи хэнтэй явах вэ? | 你和谁去呢？ |
| Хэнтэй **ч** хамаагүй. | 和谁去都没关系。 |
| | |
| Юу болсон вэ? | 发生什么了？ |
| Юу **ч** болоогүй. | 什么也没发生。 |

---

## "би ч бас" 结构

表达"我也……"时，用 **би ч бас**：

> Бид нар кинонд явна. Чи яах вэ?
> (我们要去电影院了，你要做什么？)
>
> Би **ч бас** явна.
> (我也去。)

---

## 用法总结

| 用法 | 结构 | 示例 |
|------|------|------|
| 强调"甚至" | 名词/代词 + ч | Хүүхэд ч мэднэ. |
| 强调"也" | 主语 + ч бас | Би ч бас явна. |
| "无论" | 疑问词 + ч | Юу ч болоогүй. |
| "任何都" | аль ч / хэзээ ч | Аль ч болно. |

## 常见错误

- 将 **ч** 与 **бас** 遗漏（"我也"需要 **би ч бас**）
- 混淆 **ч** 和 **бас** 的位置
