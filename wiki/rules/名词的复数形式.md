# 名词的复数形式 (Noun Plural / олон тоо)

## 规则概述

蒙古语名词复数通过添加后缀构成，后缀选择取决于词尾元音。

## 复数后缀表

蒙古语有 **7 种**复数附加成分，其中最常用的是前两种：

### 1. -нууд / -нүүд

接在以元音结尾或以辅音 **н** 结尾的名词后，表示人和事物。

| 词干 | 复数 | 含义 |
|------|------|------|
| чулуу | чулуунууд | 石头们 |
| ширээ | ширээнүүд | 桌子们 |
| байгууллага | байгууллаганууд | 机关们 |
| үйлдвэр | үйлдвэрнүүд | 工厂们 |
| дайсан | дайсэн-нууд | 敌人们 |
| эрдэмтэн | эрдэмтэн-нүүд | 学者们 |
| зочин | зочин-нууд | 客人们 |

### 2. -ууд / -үүд (口语中)

接在以辅音结尾的名词后。在以长元音、复合元音结尾的词后变为 **-гууд / -гүүд**。

| 词干 | 复数 | 含义 |
|------|------|------|
| бичиг | бичгүүд | 书籍们 |
| улс | улсууд | 国家/人们 |
| явдал | явдлууд | 事情们 |
| айл | айлууд | 村们 |
| хүүхэд | хүүхдүүд | 儿童们 |
| гэр | гэрүүд | 屋们 |
| нам | намууд | 党们 |
| үйлдвэр | үйлдвэрүүд | 一些工厂 |
| эрдэмтэн | эрдэмтэнүүд | 学者们 |

### 3. -нар / -нер

接在表示人物或拟人事物的名词后，一般分写（不连写）。有时起总括词作用。

| 词干 | 复数 | 含义 |
|------|------|------|
| ах | ах нар | 兄长们 |
| дүү | дүү нэр | 弟弟们 |
| багш | багш нар | 老师们 |
| эмч | эмч нар | 医生们 |
| найз | найз нар | 朋友们 |

### 4. -чууд / -чүүд

接在表示人的属性的形容词及民族名称后，兼有构词作用。

| 词干 | 复数 | 含义 |
|------|------|------|
| мэргэн | мэргэчүүд | 聪明人 |
| баян | баячууд | 富人 |
| ядуу | ядуучууд | 穷人 |
| монгол | монголчууд | 蒙古民族 |
| уйгар | уйгарчууд | 维吾尔民族 |

### 5. -д

主要接在以 **-н**（尤其以 **-чин**）结尾的名词后，偶尔接在以 **-р, -л, -й** 结尾的少数名词后。接加时前边语音可能脱落。

| 词干 | 复数 | 含义 |
|------|------|------|
| малчин | малчид | 牧民们 |
| тариачин | тариачид | 农民们 |
| морь | морьд | 几匹马 |
| шувуу | шувууд | 一些鸟 |
| түшмэл | түшмэд | 大臣们 |
| нохой | ноход | 几只狗 |
| хувцас | хувцад | 衣物 |

### 6. -с

接在以元音结尾的一部分名词后，偶尔接在以 **-н, -л, -й** 结尾的少数名词后。接加时有语音脱落现象。

| 词干 | 复数 | 含义 |
|------|------|------|
| эх | эхэс | 母亲们 |
| ах | ахас | 兄长们 |
| эр | эрс | 男子们 |
| эм | эмс | 女人们 |
| үр | үрс | 子孙 |
| үг | үгс | 话语 |
| уул | уулс | 山峦 |
| хүн | хүмүүс | 人们 |
| зүйл | зүйлс | 事项 |
| залуу | залуус | 青年们 |

### 7. 叠词法表示复数

名词本身重叠，或定语重叠，表示"各个""每个"的意义。

| 形式 | 示例 | 含义 |
|------|------|------|
| 名词重叠 | газар газар | 各个地方 |
| | гэр гэрт | 各个屋子 |
| 定语重叠 | өндөр өндөр уул | 一些高山 |
| | бүдүүн бүдүүн мод | 许多大树 |

## 单数形式表示复数的情况

蒙古语中并非所有复数概念都使用复数形式。以下情况常用单数形式：

1. **定语明确表示复数**：гурван хүн 三个人，олон ном 很多书
2. **动词明确表示复数**：хүн цуглаж байна 人们在集合
3. **常识可知为复数**：гутал худалдан авах 买靴子（靴子是成双的）
4. **抽象/物质/专有名词**：一般不用复数

## 常见错误

- 忽略元音和谐，错误选择后缀形式
- 将 -нар/-нер 用于所有名词（仅限特定人物名词）
- 在定语已明确表示复数时仍加复数后缀
- 混淆 -д 和 -с 的使用条件
