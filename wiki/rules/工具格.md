# 工具格 (Instrumental Case / үйлдэхийн тийн ялгал)

## 规则概述

工具格表示"用……"、"以……方式"、"凭借……"，后缀为 **-аар / -ээр / -оор / -өөр**。

> **注意**：工具格不要与[[共同格]]混淆。共同格的 "with" 是"和……一起"的意思，工具格的 "with" 是"用……"的意思。

---

## 规则详情

### 基本后缀

| 词尾元音 | 后缀 | 示例 |
|---------|------|------|
| 后元音 | -аар / -оор | хүн → хүндээр (用人) |
| 前元音 | -ээр / -өөр | ном → номоор (用书) |

### 不同词尾的变化

| 词尾类型 | 后缀 | 示例 | 含义 |
|---------|------|------|------|
| 辅音/不发音元音结尾 | -аар / -ээр / -оор / -өөр | автобус → автобус**аар** | 乘公共汽车 |
| 软音符号 ь / 短 и 结尾 | -иар | англи → англи**ар** | 用英语 |
| 隐藏 g 名词 | -гаар | номын сан → номын сан**гаар** | 通过图书馆 |
| 长元音结尾 | -гаар | морь → мори**ар** | 骑马 |

---

## 用法详解

### 1. 表示工具/手段 — "用……"

| 示例 | 含义 |
|------|------|
| Би үзг**ээр** бичсэн. | 我用铅笔写（过去时）。 |
| Дорж мод**оор** ширээ хийсэн. | 道尔吉用木头做了桌子。 |
| Би мор**иор** ирсэн. | 我骑马来的。 |
| Би сургуульдаа автобус**аар** ирсэн. | 我乘公共汽车来了学校。 |

### 2. 表示语言/媒介 — "用……语"

| 示例 | 含义 |
|------|------|
| Тэр англи**ар** их сайн ярьдаг. | 她英语说得很好。 |
| Бид монгол**оор** бага сага ярьдаг. | 我们只会说一点点蒙古语。 |

### 3. 惯用表达

工具格有一些地道的惯用方式：

| 示例 | 含义 | 说明 |
|------|------|------|
| Чамаар сонин сайхан юу байна? | （从）你有什么新鲜事？ | 问候语 |
| Хотоор сонин юу байна? | （从）城里有什么新鲜事？ | 询问消息 |
| Одоо дэлгүүрээр явах уу? | 我们现在去商店吧？ | -аар 表示去不止一家 |
| Чи өдрийн хоолоор юу юу идсэн бэ? | 你中午吃什么了？ | |

> **注意**："дэлгүүрээр явах" 中，-аар 表示要去的不止一家商店。如果说"去那家店"，则用向位格："дэлгүүрт очих уу?"

## 人称代词的工具格

| 主格 | 工具格 | 含义 |
|------|--------|------|
| би | надаар | 用我 / 由我 |
| та | танаар | 用您 / 由您 |
| чи | чимаар | 用你 / 由你 |
| тэр | түүгээр / түүнээр | 用他 / 由他 |
| бид | биднээр | 用我们 |
| та нар | та нараар | 用你们 |
| тэд | тэднээр | 用他们 |

## 反身领属与工具格

反身领属可以加在工具格后面：

| 示例 | 含义 |
|------|------|
| Би үзгээрээ бичсэн. | 我用我自己的笔写了。 |

> (他用铅笔写了。)
> — [[日常问候]] | 语法点: [[工具格]]

## 常见错误

- 与[[共同格]] (-тай) 混淆：工具格强调手段，共同格强调伴随
- 元音和谐错误
- 隐藏 g 名词忘记加 -г
