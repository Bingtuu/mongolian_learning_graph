# 属格的形成 (Genitive Case / харьяалахын тийн ялгал)

## 规则概述

属格表示所有关系，相当于汉语的"的"。属格后缀为 **-ын / -ийн / -ы / -ий / -н**。

## 规则详情

### 基本规则

| 词尾类型 | 后缀 | 示例 | 说明 |
|---------|------|------|------|
| 元音/双元音/长 и 结尾 | -н | ширээ → ширээний (桌子的) | N-stem 名词 |
| 后元音 + -н 结尾 | -ы | мах → махы (肉的) | N-stem |
| 前元音 + -н 结尾 | -ий | морь → морий (马的) | N-stem，软音符号 ь 脱落 |
| 前元音/辅音(除 -н) | -ийн | дэвтэр → дэвтрийн (笔记本的) | 短元音脱落 |
| 后元音/辅音(除 -н, -ж, -ч, -ш, -г, -ь, -и) | -ын | ном → номын (书的) | |

### N-stem 名词的属格

N-stem 名词（词尾为 -н 的名词）在属格中保留或添加 -н：

| 原形 | 属格 | 说明 |
|------|------|------|
| өрөө | өрөөний | 元音 + н |
| мах | махы | 辅音 + а + н |
| морь | морий | 软音符号 ь 脱落，加 -ий |
| хэл | хэлний | 辅音 + э + н |
| ус | усны | 辅音 + а + н |

### 代词的属格

| 人称 | 属格 |
|------|------|
| би (我) | миний |
| чи (你，非敬) | чиний |
| та (您) | таны |
| тэр (他/她) | түүний |
| бид (我们) | бидний / манай |
| та нар (你们) | та нарын / танай |
| тэд (他们) | тэдний |

> **说明**：бид 和 та нар 的属格有两种形式：бидний/манай 和 та нарын/танай。манай 和 танай 是古变格的残存形式，现在已不做严格区分。

## 例句

> Энэ миний **номын** хуудас.
> (这是我的书的页面。)
> — [[自我介绍]] | 语法点: [[属格的形成]]

> Баатарын морь
> (Baatar 的马)
> — [[日常问候]] | 语法点: [[属格的形成]]、[[N-stem 名词]]

> Манай ахын эхнэр.
> Manai akhyn ekhner.
> манай / ахын / эхнэр
> 我的 / 哥哥的 / 妻子
> (我哥哥的妻子。)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[属格的形成]]、[[名词谓语]]

> Миний ээжийн ээж миний эмээ.
> Minii eejiin eej minii emee.
> миний / ээжийн / ээж / миний / эмээ
> 我的 / 母亲的 / 母亲 / 我的 / 祖母
> (我母亲的母亲是我的祖母。)
> — 来源: Peace Corps Unit 2 Lesson 1 p.38 (Exercise 3) | 语法点: [[属格的形成]]、[[名词谓语]]

## 常见错误

- 混淆 -ын 和 -ийн 的使用条件
- 忘记元音和谐对后缀选择的影响
- N-stem 名词忘记保留/添加 -н
