# 强调句 (Emphasis / онцлох)

## 规则概述

蒙古语中可以通过在动词后加**双元音**（-аа / -ээ / -оо / -өө）来构成强调句。这个双元音与动词**分开写**，不连写成一个词。

---

## 动词后加双元音

在陈述句或答句中，将双元音放在动词后面，表达肯定、确认或感叹的语气：

| 示例 | 含义 |
|------|------|
| Хүйтэн байна **аа**! | 真冷啊！ |
| Явна **аа**, явна. | 要去的，要去。（肯定回答） |
| Болно **оо**, болно. | 可以的，可以。（肯定回答） |

> **注意**：双元音**必须和动词分开写**，不能连成一个词。

---

## 与疑问句联用

在回答疑问句时，用双元音加强肯定语气：

| 对话 | 含义 |
|------|------|
| — Чи сургуульдаа явах уу? | — 你要去学校吗？ |
| — Явна **аа**, явна. | — 要去的，要去。 |
| | |
| — Үүнийг авч болох уу? | — 我能拿这个吗？ |
| — Болно **оо**, болно. | — 可以的，可以。 |

---

## 感叹用法

表达感叹时，双元音常带有强烈的感情色彩：

| 示例 | 含义 |
|------|------|
| Хүйтэн байна **аа**! | 真冷啊！ |
| Яасан сайн хүү вэ! | 多么好的孩子啊！ |

---

## 与 "за" 连用

**за**（行、好）常与强调结构连用，表示同意或确认：

| 示例 | 含义 |
|------|------|
| За, одоо сургуульдаа яв! | 好，现在去学校吧！ |

---

## 用法总结

| 场景 | 结构 | 示例 |
|------|------|------|
| 肯定回答 | 动词 + аа/ээ/оо/өө（分开写） | Явна аа, явна. |
| 感叹 | 形容词/动词 + аа | Хүйтэн байна аа! |
| 确认 | болно оо / болноо | Болно оо, болно. |

## 常见错误

- 将双元音与动词连写（应分开写）
- 在不需要强调的句子中滥用强调结构
