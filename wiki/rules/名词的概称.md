# 名词的概称 (Appellative / ерөнхий нэр)

## 规则概述

概称使某个事物的涵义变得广泛，将本来指具体事物的词变为可以包括同类或密切相关事物的词。相当于汉语的"……等""……什么的"。

## 构成方式

### 基本规则

在一个名词（主体词）后重复一个以 **"м"** 辅音开头的同样词（辅助词）：

1. **以辅音为首的名词**：辅助词把原词首辅音换成 **"м"**
2. **以元音为首的名词**：辅助词在原词首加 **"м"**

### 示例

| 原形 | 概称 | 含义 |
|------|------|------|
| уул | уул муул | 山什么的 |
| авдар | авдар мавдар | 箱子什么的 |
| эмч | эмч мэмч | 医生什么的 |
| ус | ус мус | 水什么的 |
| үхэр | үхэр мүхэр | 牛什么的 |
| онгоц | онгоц монгоц | 船什么的 |
| цай | цай май | 茶什么的 |
| гутал | гутал мутал | 靴子什么的 |
| давс | давс мавс | 盐什么的 |
| тариа | тариа мария | 田地什么的 |
| нохой | нохой мохой | 狗什么的 |
| шүлэг | шүлэг мүлэг | 诗什么的 |
| бүжиг | бүжиг мүжиг | 舞蹈什么的 |

### 特殊形式

以 **"м"** 为首的名词概称时，使用其他辅音（规律性不强，当作词汇单位处理）：

| 原形 | 概称 | 含义 |
|------|------|------|
| ах | ах зах | 兄长等 |
| малгай | малгай жалгай | 帽子等 |
| машин | машин жашин | 机械等 |
| урам | урам турам | 情绪等 |
| мод | мод шод | 木头等 |
| хүүхэд | хүүхэд шуухад | 小孩等 |
| юм | юм хүм | 东西等 |

### 方言变体

部分地区用其他方式表达概称：

| 形式 | 示例 | 含义 |
|------|------|------|
| юу | малгай юугаа | 帽子等 |
| амтай | будаа амтай | 饭菜等 |

## 用法

概称主要在口语中使用，作用在于：
- 使语言具有灵活性
- 有修辞及语音协调作用
- 有时带有"贬称"或不够郑重的意味

> 正式场合可用近义词重复代替，避免不够郑重的副作用。

## 例句

> Аягаа хураа → **Аяга маягаа** хураа.
> (把碗收拾起来 → 把碗等东西收拾起来。)
