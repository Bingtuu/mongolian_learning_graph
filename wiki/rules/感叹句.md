# 感叹句 (Exclamatory Sentence / аялга өгүүлбэр)

## 规则概述

感叹句表示强烈的感情。句末语调**下降**，书面上用**感叹号**表示。

> 来源: 清格尔泰语法 2.2.2.4

---

## 规则详情

### 1. 常用句式：陈述句 + 特殊疑问语气词

最常见的感叹句句式是**陈述句与特殊疑问句的混合**：在陈述句句末加上特殊疑问语气词，再把疑问代词加在所要强调的词前边。

| 句式结构 | 说明 | 示例 |
|----------|------|------|
| **ямар / хэдий** + 形容词/动词 + **вэ** | 感叹性质或程度 | Ямар сайхан өдөр вэ! |
| **хэдий** + 形容词 + 名词 + **бэ** | 感叹事物的特征 | Хэдий өндөр мод бэ! |
| 陈述句 + **доо / шүү** + 感叹号 | 用陈述句加强语气 | Аштай сайн боллоо доо! |

### 2. 各类句式的感叹化

祈使句、疑问句、陈述句，如果用**强烈的感情**来说时，也可以变为感叹句。感叹句中**强调的重点说得特别重**，常常使用**感叹词**和**呼语**。

| 原句式 | 感叹化示例 | 含义 |
|--------|-----------|------|
| 陈述句 | Энэ хэргийг би насандаа мартахгүй! | 这个事情我一辈子也忘不了！ |
| 疑问句 | Хүүе номын дэлгүүрт арвин шинэ ном ирж гэнэ шүү! | 嘿，听说书店里来了很多新书！ |
| 祈使句 | Нөхөд өө бид ажлаа түргэн дуусгая! | 同志们，我们快把工作干完吧！ |

### 3. 感叹词与呼语

感叹句中常使用**感叹词**（аялга үг）和**呼语**（дуудах үг）来增强感情色彩。

#### 常见感叹词

| 感叹词 | 含义 | 示例 |
|--------|------|------|
| **хүүе** | 喂、嘿（惊讶、唤起注意） | Хүүе тэр чинь юу юм бэ? |
| **уухай** | 唉、啊（赞叹、感慨） | Уухай яг таарч байна. |
| **ай даа** | 唉呀（惆怅、遗憾） | Ай даа хэдий ч маягтай юм. |
| **ай** | 唉（惊讶、不满） | Ай чи яагаад оройтчихсон бэ? |
| **ва лам аа** | 天啊 | Ва лам аа! |
| **ёо ёо** | 哎呀 | Ёо ёо! |

#### 呼语的使用

呼语浮游于句子结构之外，表示呼唤、提醒。

> Нөхөд **өө** дөрвөн мөрөөр зогсоцгоо!
> 同志们，排四行站！

> **Бат гуай** чөлөөгөөрөө манайд тойрч ир дээ.
> 巴图先生！有空到我们这里串门。

## 例句

> Ха яасан сайхан хэлж байх вэ!
> Ha yasan saikhan khelj baikh ve!
> ха / яасан / сайхан / хэлж / байх / вэ
> 哈 / 多么 / 好 / 说 / 正在 / (感叹)
> (哈，说得多么好呀！)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

> Хэдий өндөр мод бэ энэ яаж ургасан бэ!
> Khedii öndör mod be ene yaaj urgasan be!
> хэдий / өндөр / мод / бэ / энэ / яаж / ургасан / бэ
> 多么 / 高 / 树 / 呀 / 这 / 怎么 / 生长 / 呀
> (多高的树呀！怎么生长的呀！)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

> Ямар муухай өдөр вэ!
> Yamar muukhai ödör ve!
> ямар / муухай / өдөр / вэ
> 多么 / 不好 / 天 / 呀
> (天气真不好！)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

> Энэ юмыг хэдий сайхан хийсэн бэ!
> Ene yumiig khedii saikhan khiisen be!
> энэ / юмыг / хэдий / сайхан / хийсэн / бэ
> 这 / 东西(宾格) / 多么 / 好 / 做 / 呀
> (这个东西作得多好呀！)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

> Аштай сайн боллоо доо.
> Ashtai sain bolloo doo.
> аштай / сайн / боллоо / доо
> 终于 / 好 / 变成了 / (感叹)
> (这回好了！)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

> Тэр хүн явбал явмайч чамд ямар хамаа байх юм!
> Ter khün yavbal yavmaich chamd yamar khamaa baikh yum!
> тэр / хүн / явбал / явмайч / чамд / ямар / хамаа / байх / юм
> 那 / 人 / 走 / 就走呗 / 对你 / 什么 / 关系 / 有 / 呢
> (他走就走呗，和你有什么关系?!)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

> Маргааш би бас явбал аа!
> Margaash bi bas yaval aa!
> маргааш / би / бас / явбал / аа
> 明天 / 我 / 也 / 去 / 啊
> (明天我也去多好啊！)
> — 来源: 清格尔泰语法 2.2.2.4 | 语法点: [[感叹句]]

## 感叹句 vs 其他句式

| 对比项 | 感叹句 | 祈使句 | 疑问句 |
|--------|--------|--------|--------|
| 语调 | **下降** | 下降 | 升高 |
| 标点 | **!** | ! 或 . | ? |
| 核心功能 | 表达强烈感情 | 要求、命令、祝愿 | 询问、怀疑 |
| 可转化 | 可由陈述/疑问/祈使句转化 | 一般不能转化 | 一般不能转化 |
