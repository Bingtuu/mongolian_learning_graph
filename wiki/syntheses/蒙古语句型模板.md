# 蒙古语句型模板 (Sentence Patterns / өгүүлбэрийн загвар)

> 本页提供蒙古语最常用的句型模板。掌握这些模板后，只需替换其中的词汇，就能造出大量正确的句子。

---

## 模板 1：名词谓语句 — "……是……"

```
[主语] + [名词/形容词]
```

| 模板 | 示例 |
|------|------|
| Би + [身份]. | Би багш. (我是老师。) |
| Тэр + [身份]. | Тэр оюутан. (他是学生。) |
| Энэ + [事物]. | Энэ ном. (这是书。) |
| [事物] + [形容词]. | Энэ өрөө том. (这个房间大。) |

**否定**：
```
[主语] + [名词/形容词] + биш.
```
> Би багш биш. (我不是老师。)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

> 详见 [[名词谓语]]。

---

## 模板 2：动词句 — "……做……"

```
[主语（主格）] + [宾语（宾格，可选）] + [动词（时态后缀）]
```

| 时态 | 模板 | 示例 |
|------|------|------|
| 现在/将来 | [主] + [宾] + [动词]-на | Би ном унш**ина**. |
| 过去 | [主] + [宾] + [动词]-сан | Би ном унш**сан**. |
| 习惯 | [主] + [宾] + [动词]-даг | Би ном унш**даг**. |
| 进行 | [主] + [宾] + [动词]-ж байна | Би ном унши**ж байна**. |

**注意**：宾语可以省略宾格标记，尤其当宾语是泛指时。

---

## 模板 3："有……"句型

### 用共同格（强调"有什么"）

```
[主语（主格）] + [事物] + -тай/-тэй.
```

> Би найз**тай**. (我有朋友。)
> Тэр мөнгө**тэй**. (她有钱。)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

### 用给格（强调"谁有"）

```
[所有者（给格）] + [事物（主格）] + байна.
```

> **Надад** ном байна. (我有书。)
> **Чамд** хэдэн ном байна вэ? (你有多少本书？)

> 详见 [[共同格]]、[[给格]]。

## 模板 4：比较句 — "A 比 B……"

```
[A（主格）] + [B（离格）] + [形容词].
```

> Чи над**аас** өндөр. (你比我高。)
> Би чам**аас** залуу. (我比你年轻。)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

> 详见 [[离格]]。

## 模板 5：方位句 — "……在……"

```
[事物（主格）] + [位置（给格/位格）] + байна.
```

> Аав гэр**т** байна. (爸爸在家。)
> Ном ширээ**нд** байна. (书在桌子上。)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

> 详见 [[给格]]、[[位格]]。

## 模板 6：一般疑问句 — "……吗？"

```
[陈述句] + уу/үү/юу?
```

> Энэ ном **уу**? (这是书吗？)
> Та багш **үү**? (您是老师吗？)
> Би очих **уу**? (我要去吗？)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

**回答**：
- 是 → **Тийм**.
- 否 → **Үгүй**.

> 详见 [[疑问句]]。

## 模板 7：特殊疑问句 — "谁/什么/哪里……？"

```
[疑问词] + [句子其他部分] + бэ/вэ?
```

| 想问 | 疑问词 | 示例 |
|------|--------|------|
| 谁 | хэн | **Хэн** явах вэ? (谁会去？) |
| 什么 | юу | Та **юу** идэж байна вэ? (您在吃什么？) |
| 哪里 | хаана | Чи **хаана** байна вэ? (你在哪里？) |
| 什么时候 | хэзээ | Чи **хэзээ** ирэх вэ? (你什么时候来？) |
| 多少 | хэдэн | Чамд **хэдэн** ном байна вэ? (你有多少本书？) |
| 什么样的 | ямар | **Ямар** мах сайхан бэ? (什么样的肉好吃？) |

> 详见 [[疑问句]]、[[代词]]。

## 模板 8：祈使句 — "请……""别……"

### 肯定命令

```
[动词词干]!
```

> Яв! (走！) / Суу! (坐！) / Унш! (读！)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

### 礼貌请求

```
Та + [动词词干]!
```

> Та яв! (请您走吧！) / Та суу! (您请坐！)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

### 否定命令

```
Бүү / Битгий + [动词词干]!
```

> Бүү яв! (别走！) / Битгий яв! (别走！)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

> 详见 [[祈使式]]。

## 模板 9：意愿句 — "让我们……"

```
[主语] + [动词]-ъя/-ье/-я.
```

> Би хий**е**. (我来做吧！) / Яв**ъя**! (我们走吧！)

## 模板 10：条件句 — "如果……就……"

```
[条件（-вал/-вэл）] + [结果].
```

> Маргааш бороо ор**вол** явлагагүй. (如果明天下雨就不要走了。)
> Чи хэл**вэл** би мэднэ. (如果你说我就知道。)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

> 详见 [[条件式]]。

## 模板 11：进行中的伴随动作

```
[动作1]-ж + [动作2].
```

> Тэр ном унши**ж** суув. (他坐着读书。)
> Би дуу сонсо**ж** явна. (我边走边听音乐。)
> — 来源: [[Gaunt 现代蒙古语]] | 语法点: [[蒙古语句型模板]]

> 详见 [[副动词]]。

## 模板组合示例

将多个模板组合，可以表达更复杂的意思：

> **Өнөөдөр** (时间) **би** (主语) **найзтайгаа** (共同格+反身领属) **хамт** (一起) **кино үзэхээр** (目的副动词) **сургууль руу** (向格) **явна.** (谓语)
> 
> (今天我要和朋友一起，为了看电影，向学校去。)
> 
> → 自然翻译：今天我要和朋友一起去学校看电影。
