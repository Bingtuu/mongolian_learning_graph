# 初学者语法路线图 (Beginner's Grammar Roadmap / эхлэн суралцагчдын замнал)

> 本页为蒙古语初学者提供一条从"零基础"到"能进行简单对话"的渐进式学习路径。每个阶段标注了必须先掌握的核心知识点和可选的扩展内容。

---

## 阶段 1：发音与文字（第 1-3 天）

**目标**：能读出任何西里尔蒙文单词，理解基本发音规则。

### 必学内容

| 知识点 | 页面 | 重要性 |
|--------|------|--------|
| 西里尔字母表 | [[蒙古语字母表]] | ⭐⭐⭐ 基础中的基础 |
| 元音分类（前后/中性） | [[元音分类]] | ⭐⭐⭐ 理解元音和谐的前提 |
| 元音和谐律 | [[元音和谐律]] | ⭐⭐⭐ 决定所有后缀形式 |

### 阶段 1 自测

> 能正确读出：Монгол хэл сонирхолтой. (蒙古语很有趣。)
> 能判断单词的元音类型（阳性/阴性）。

练习页：[[阶段1-字母与问候|阶段1：字母与问候练习]]、[[元音和谐-后缀选择|元音和谐：后缀选择练习]]

## 阶段 2：基础句型（第 4-7 天）

**目标**：能用简单句介绍自己、表达"是/有/做"。

### 必学内容

| 知识点 | 页面 | 重要性 |
|--------|------|--------|
| 名词谓语句（不用"是"） | [[名词谓语]] | ⭐⭐⭐ 最基础的句型 |
| 主格与宾格 | [[名词变格]] | ⭐⭐⭐ 句子的骨架 |
| 现在/将来时 (-на) | [[现在时构成]] | ⭐⭐⭐ 第一个动词时态 |
| 人称代词 | [[代词]] | ⭐⭐⭐ 每天都在用 |
| 否定式基础 | [[否定式]] | ⭐⭐ 学会说"不" |
| 一般疑问句 | [[疑问句]] | ⭐⭐ 学会提问 |

### 阶段 2 自测

> 能说出完整的自我介绍：
> Сайн байна уу? Би ... Би ... хүн. Би ... сурна.

> 能进行简单问答：
> — Энэ ном уу? — Тийм, энэ ном. / Үгүй, энэ ном биш.

练习页：[[阶段2-自我介绍|阶段2：自我介绍练习]]

| 现在进行时 (-ж байна) | [[现在时构成]] | ⭐⭐⭐ 表达正在进行 |
| 时态对比 | [[时态对比]] | ⭐⭐⭐ 选择正确的时态 |
| 祈使式 | [[祈使式]] | ⭐⭐ 命令/请求 |

### 选学内容

| 知识点 | 页面 | 说明 |
|--------|------|------|
| 副动词 (-ж) | [[副动词]] | 连接动词 |
| 意愿式 (-ъя) | [[愿望式]] | "让我们……" |

### 阶段 4 自测

> 能用不同时态描述同一动作：
> - 现在：Би ном уншина. (我要读书。)
> - 进行：Би ном уншиж байна. (我正在读书。)
> - 习惯：Би ном уншдаг. (我常读书。)
> - 过去：Би ном уншсан. (我读过书。)

## 学习建议

### 每日学习节奏

| 时间 | 内容 |
|------|------|
| 10 分钟 | 复习前一天的知识点 |
| 20 分钟 | 学习新语法点 |
| 10 分钟 | 朗读例句（出声！） |
| 10 分钟 | 尝试造 3-5 个自己的句子 |

### 避免的学习陷阱

1. **不要急于记所有格的变化** — 先掌握主格、属格、给格、宾格，其他的随用随查。
2. **不要忽视发音** — 元音和谐律不只是理论，它决定了你说话是否地道。
3. **不要只看不练** — 蒙古语是黏着语，后缀变化必须亲手变位才能记住。
4. **不要跳过例句** — 每个语法点都要配合例句理解。

### 推荐学习顺序速查

```
字母表 → 元音和谐律 → 名词谓语 → 主格/宾格 → 现在时(-на)
    ↓
属格/给格 → 过去时(-сан) → 习惯式(-даг) → 进行时(-ж байна)
    ↓
形容词 → 数词 → 副词/后置词 → 否定式/疑问句
    ↓
被动/使役 → 副动词 → 形动词 → 句法/构词法
```
