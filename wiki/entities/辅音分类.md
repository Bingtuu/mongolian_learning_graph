# 辅音分类 (Consonant Classification / гийгүүлэгч)

## 概述

蒙古语西里尔字母中有 20 多个辅音字母，按发音方式可分为塞音、擦音、塞擦音、鼻音、边音、颤音等。

## 辅音分类表

| 发音方式 | 字母 |
|---------|------|
| 塞音 | б, п, д, т, г, к |
| 擦音 | в, с, з, ш, х, ф |
| 塞擦音 | ж, ц, ч |
| 鼻音 | м, н, нг |
| 边音 | л |
| 颤音 | р |
| 半元音 | й |

## 说明

- 辅音和谐也是蒙古语的重要特征（参见 [[元音和谐律]]）
- 部分辅音有清浊对立（如 б/п, д/т, г/к）
