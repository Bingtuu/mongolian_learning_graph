# 元音分类 (Vowel Classification / эгшиг)

## 概述

蒙古语共有 7 个基本元音字母，按发音位置分为后元音、前元音和中性元音三类。

## 元音分类表

| 类型 | 字母 | 发音（近似） |
|------|------|------------|
| **后元音** | А а | [a] |
| | О о | [ɔ] |
| | У у | [ʊ] |
| **前元音** | Э э | [ɛ] |
| | Ө ө | [œ] |
| | Ү ү | [u] |
| **中性元音** | И и | [i] |

## 说明

- 中性元音 **и** 可与前后元音共存，不影响和谐律
- 元音和谐律决定了后缀中元音的选择
