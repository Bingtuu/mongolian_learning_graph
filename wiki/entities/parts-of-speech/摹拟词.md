# 摹拟词 (Onomatopoeia / дуу дүрслэх үг)

## 概述

摹拟词摹仿声音或形象地表达行为状态。分为摹声词和拟态词。摹拟词的特点是直接摹拟声音或形象地表达行为状态，一般采取词干重叠的形式，有时后边用关联动词 **гэх** 来连接。

## 语法特征

- 没有形态变化
- 在句中一般充当状语成分
- 有时可借助 гэх 充当其他成分
- 采取词干重叠的形式

## 分类

### 1. 摹声词 (дуу дуулах үг)

摹仿各种声音。

| 摹声词 | 含义 | 示例 |
|--------|------|------|
| ха ха | 哈哈（笑声） | ха ха гэтэл инээнэ (哈哈地笑) |
| ван ван(г) | 汪汪（狗叫） | нохой ван ван гэж хуцна |
| гангар гонгор | 雁叫声 | галуу гангар гонгор дуу гарган ниснэ |
| дүн дүн | 鼓声 | |
| пис пас | 拍手声 | |
| тин тан(г) | 枪声 | буугийн дуу тин тан дуугарна |
| хүр хүр | 火势声 | нүүрс хүр хүр гэтэл ноцож байна |
| гор гор | 小溪声 | горхины ус гор гор гэж урсана |
| тар тар | 踏硬地声 | |
| шивэр шивэр | 细雨声 | бороо шивэр шивэр гэж орж байна |
| сэр сэр | 和风声 | сэрүүн салхин сэр сэр үлээнэ |

> Бороо **шивэр шивэр** гэж орж байна.
> (雨在淅淅沥沥地下。)

> Зуухны доторхи нүүрс **хүр хүр** гэтэл ноцож байна.
> (灶里的煤呼呼地燃着。)

### 2. 拟态词 (байдал дүрслэх үг)

形象地表达行为状态。

| 拟态词 | 含义 | 示例 |
|--------|------|------|
| алцага алцага | 蹒跚 | |
| дайван дайван | 颠簸 | |
| түнтэр тантар | 跌跌撞撞 | |
| доголос доголос | 一瘸一拐 | доголос доголос гэж явж байна |
| бөмбөр бөмбөр | 来回滚动 | |
| луг луг | 怦怦（脉搏） | цусны судал нь луг луг гэж цохилж байна |
| гялас гялас | 闪闪（发光） | гялас гялас гэж цахилгаан цахилж |
| бөн бөн | 一颠一颠 | |

> Хөл нь шархдаад **доголос доголос** гэж явж байна.
> (他腿受了伤，一瘸一瘸地走着。)

> Цусны судал нь **луг луг** гэж цохилж байна.
> (血管怦怦地搏动。)

## 与 гэх 的搭配

摹拟词借助联系动词 **гэх** 实现各种变化，充当不同句子成分。

| 形式 | 功能 |
|------|------|
| гэж | 连接动词 |
| гэтэл | 时间连接 |
| гэсээр | 延续 |

> Жүжиг үзэж байгаад **ха ха гэтэл** инээнэ.
> (看着戏哈哈地笑了。)
