# 动词 (Verb / үйл үг)

## 概述

动词表示动作、状态或变化。蒙古语动词有丰富的时态、体和式的变化，是蒙古语语法中最复杂的词类。

## 特点

| 特征 | 说明 |
|------|------|
| 时态 | 现在时、过去时、将来时 |
| 体 | 完成体、进行体、持续体等 |
| 式 | 陈述式、祈使式、愿望式等 |
| 态 | 主动态、被动态、互动态、使动态、共动态、众动态 |

## 动词分类

蒙古语动词从意义上可分为两大类：

### 1. 实义动词 (实义动词 / утгат үйл үг)

有具体词汇意义的动词，表示实际的动作或状态。

- **及物动词** (transitive)：需要宾语，意义才完整。如： үзэх (看)、бичих (写)、идэх (吃)。
- **不及物动词** (intransitive)：不需要宾语，意义已完整。如： унтах (睡)、явах (走)、харайх (跳)。
- 有些动词兼有及物和不及物两种性质。如： явах (走) → зам яв (走路)。

### 2. 虚义动词 (虚义动词 / утгагүй үйл үг)

词汇意义较虚，主要起语法作用的动词。包括：

- **代动词**：代替实义动词，如 ингэх (这样做)、чингэх (那样做)。
- **关联动词 гэх**：连接思维和言语内容，是蒙古语特有的语法现象。
- **助动词**：辅助实义动词表示体、情态、趋向等语法意义。如 байх、болох 等。

详见 [[实义动词]]、[[虚义动词]]、[[助动词]]。

## 动词词干层次

蒙古语动词词干可分为四个层次：

1. **一次词干**：去掉式动词、形动词、副动词后缀后的基本形式。
2. **二次词干**：在一次词干上加体后缀构成。
3. **态性词干**：在词干上加态后缀构成。
4. **体性词干**：在态性词干上加体后缀构成。

## 动词的主要语法范畴

### 态的范畴 (Voice / бэлэгдэл)

表示行为与其主体之间的关系。蒙古语有 **六种态**：

1. **自动态** — 无专门后缀，主体主动发出动作。
2. **使动态** — 后缀 -уул/-үүл, -лга/-лгэ, -га/-гэ，表示令他人做某事。
3. **被动态** — 后缀 -гд/-д，表示主体遭受动作。
4. **互动态** — 后缀 -лд，表示互相动作。
5. **共动态** — 后缀 -лц，表示一起动作。
6. **众动态** — 后缀 -цгаа/-цгээ，表示众人一起做某事。

详见 [[动词的态范畴]]。

### 体的范畴 (Aspect / хэв)

表示行为在其发展的哪一阶段，具有什么特殊性质。

1. **普通体** — 无专门形式。
2. **暂短体** — 后缀 -схий，表示短暂动作。
3. **完成体** — 后缀 -чих，表示动作已完成。
4. **请求完成体** — 后缀 -аадах/-ээдэх，表示希望完成。
5. **分析形式体** — 副动词 + 助动词 байх。
6. **迭词形式体** — 动词重叠表示持续、继续、多次等意义。

详见 [[动词的体范畴]]。

### 式的范畴 (Mood / маяг)

表示说话人对行为的态度。详见各式的专门页面：

- [[祈使式]] — 命令、希望、提醒
- [[陈述式]] — 过去时、现在将来时、临界体
- [[条件式]] — 假设条件
- [[愿望式]] — 表达愿望

## 动词形式

### 式动词 (Finite Verb)

可独立作谓语的动词形式，有人称和时的变化。

### 形动词 (Participle / хэлэхүүн)

兼具动词性和名词性，可充当各种句子成分。详见 [[形动词]]。

### 副动词 (Converb / дүгнэхүүн)

连接动词与动词的形式，表示各种逻辑关系。详见 [[副动词]]。
