# 感情词 (Interjection / сэтгэл хөдлөл үг)

## 概述

感情词直接表示某种感情、感觉，或者表示答应、呼唤的声音。感情词没有形态变化，一般在句外单独使用，不参加句子结构。但在特殊情况下，借助联系动词 **гэх** 可以构成句子成分。

## 分类

### 1. 表达感情或感觉

| 感情词 | 含义 | 示例 |
|--------|------|------|
| ха | 惊奇 | **Ха**, тэр нисэх онгоц хэдий хурдан юм. |
| ва лам аа | 惊奇 | **Ва лам аа**, хэдий ч арвин байх юм. |
| жахай | 合心 | **Жахай** таарч байна. |
| уухай | 合心 | **Уухай** чиний хэлсэн яг таарна. |
| пай | 后悔 | **Пай** хэрэггүй оччихсон. |
| ий | 讨厌 | **Ий** энэ чинь хэдий муухай юм. |
| оо | 醒悟 | **Оо** тийм үү? |
| ёо | 疼痛 | **Ёо** нуруу өвдөж байна. |
| тий | 嫌冷 | **Тий тий** сүрхий хүйтэн байна даа. |
| халхай | 嫌热 | **Халхай халхай** яасан халуун юм. |

> **Ха**, тэр нисэх онгоц хэдий хурдан юм.
> (哈，那个飞机多快呀。)

> **Ёо** нуруу өвдөж байна.
> (哎哟，腰痛呢。)

### 2. 表达呼唤答应

| 感情词 | 含义 | 示例 |
|--------|------|------|
| хөөе | 呼叫 | **Хөөе** нааш ир! |
| хая | 阻止 | **Хая** чи бүү ор. |
| май | 交待 | **Май** энийг чамд өгье! |
| өөе | 答应 | **Өөе** би энд байна. |
| за | 同意 | **За** чиний хэлснээр хийе. |
| аль | 伸手要 | **Аль** надад нэгийг өг. |

> **Хөөе** нааш ир!
> (喂，往这儿来！)

> **За** чиний хэлснээр хийе.
> (好，按照你说的办。)

### 3. 多义感情词

#### аа

| 含义 | 示例 |
|------|------|
| 赞同 | **Аа** энэ харин болно. |
| 喜爱赞美 | **Аа** хөөрхий дээ. |
| 后悔醒悟 | **Аа** би очсон бол таарах байжээ. |
| 奇怪 | **Аа** сонин даа. |
| 反问 | **Аа** чи юу гэнэ? |

> **Аа** яасан сайхан үзэмж бэ.
> (啊，多么优美的风景啊！)

#### ай

| 含义 | 示例 |
|------|------|
| 惊讶 | **Ай** үнэхээр санаанд үгүй хэрэг. |
| 懊悔 | **Ай** би яагаад мартчихсан юм. |
| 责备 | **Ай** чи яагаад оройтчихсон бэ? |
| 不同意 | **Ай** тэр яагаад болох юм. |

> **Ай** чи яагаад оройтчихсон бэ?
> (唉，你为什么迟到了？)

### 4. 诗歌中的呼语

诗歌、颂歌、歌谣、童谣等文学形式里用的呼语。

| 呼语 | 用途 |
|------|------|
| аа ха хөөе | 民歌衬词 |
| хүрий хүрий | 民歌衬词 |
| бүүвэй бүүвэй | 哄孩子 |
| бөмбөрэй бөмбөрэй | 民歌衬词 |

> Арван тавны сарыг хөө **бөмбөрэй**
> (把十五的月亮呀，本博来)
