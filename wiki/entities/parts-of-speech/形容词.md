# 形容词 (Adjective / тэмдэг нэр)

## 概述

形容词用于修饰名词，表示性质、状态或特征。蒙古语形容词通常位于被修饰名词之前。

## 特点

| 特征 | 说明 |
|------|------|
| 位置 | 通常位于名词之前 |
| 比较级 | 通过后缀或辅助词构成 |
| 与名词的关系 | 形容词可与名词性化后缀结合 |

## 比较范畴

### 1. 原级

形容词的基本形式，不附加比较后缀。

> Энэ нарийн шүү.
> (这个很细。)

### 2. 比较级

表示"较……"、"更……"。通过在形容词后加 **-дүү / -дүүгэй / -дүүгээр** 等后缀构成。

| 形式 | 说明 | 示例 |
|------|------|------|
| -дүү | 比较级后缀 | сайн (好) → сайн**дүү** (较好) |
| -дүүгэй | 较……一点 | жижиг (小) → жижиг**дүүгэй** (较小) |
| -дүүгээр | 以较……的方式 | | |

> Энэ тэрнээс сайн**дүү**.
> (这个比那个较好。)

### 3. 最高级

表示"最……"。通过加 **хамгийн** (最) 或在形容词后加 **-гүй / -гүйц** 等构成。

| 形式 | 说明 | 示例 |
|------|------|------|
| хамгийн + 形容词 | 最…… | хамгийн сайн (最好) |
| -гүй / -гүйц | 极尽…… | олон (多) → олон**гүй** (极多) |

> Хамгийн өндөр уул.
> (最高的山。)

### 4. 比较对象

表示比较时，被比较的对象用 **从比格**（-аас/-ээс）。

> Тэр надаас өндөр.
> (他比我高。)

## 数的变化

形容词本身没有数的范畴，但当它名词化后，可以像名词一样有复数形式。

> Сайн**ууд** нь үлдэв.
> (好的那些留下了。)

## 格的变化

形容词可以像名词一样变格，尤其当它名词化时。

| 格 | 形式 | 示例 |
|----|------|------|
| 主格 | 原形 | сайн (好的) |
| 属格 | -ын/-ийн | сайн**ыг** |
| 宾格 | -ыг/-ийг | сайн**ыг** |
| 给在格 | -д/-т | сайн**д** |
| 从比格 | -аас/-ээс | сайн**аас** |
| 用让格 | -аар/-ээр | сайн**аар** |
| 和同格 | -тай/-тэй | сайн**тай** |

## 领属关系

形容词可以附加领属附加成分，表示"某人的……"。

> Чиний **сайн** чинь яг таарна.
> (你的好正合适。)
