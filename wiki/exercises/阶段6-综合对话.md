# 阶段6：综合对话练习

> 本练习覆盖初学者语法路线图阶段6的进阶语法点：被动语态、使役式、条件式、副动词、形动词、动词的态与体。题目为综合对话形式，需灵活运用多种语法结构。

---

## 一、被动语态（-гд-）

将下列主动句改为被动句。

### 题目

1. Багш ном уншина.（老师读书。）→ Ном _______.
2. Ээж хоол хийсэн.（母亲做了饭。）→ Хоол _______.
3. Малчин хонь хариулж байна.（牧民在放羊。）→ Хонь _______.
4. Тэр бичлэг бичсэн.（他写了记录。）→ Бичлэг _______.

### 答案

1. **багшаар уншигдана** — Ном багшаар **уншигдана**.（书被老师读。）
2. **ээжээр хийгдсэн** — Хоол ээжээр **хийгдсэн**.（饭被母亲做了。）
3. **малчинаар хариулуулж байна** — Хонь малчинаар **хариулуулж байна**.（羊被牧民放。）
4. **тэрээр бичигдсэн** — Бичлэг тэрээр **бичигдсэн**.（记录被他写了。）

### 解析

- 被动语态后缀：**-гд- / -гд-**（插入动词词干）
- 被动句中，原宾语变主语，原主语用工具格表示施动者
- 结构：受动者（主格）+ 施动者（工具格）+ 动词被动形式

## 二、使役式（-уул- / -лга-）

用使役式改写下列句子。

### 题目

1. Би хоол хийсэн.（我做了饭。）→ Би хүүхдээр _______.
2. Тэр ном уншсан.（他读了书。）→ Би түүнийг _______.
3. Бид дуу дуулсан.（我们唱了歌。）→ Бид хүүхдүүдээр _______.
4. Тэр захиа бичсэн.（他写了信。）→ Тэр надаар _______.

### 答案

1. **хоол хийлгэсэн** — Би хүүхдээр хоол **хийлгэсэн**.（我让孩子做了饭。）
2. **ном уншуулсан** — Би түүнийг ном **уншуулсан**.（我让他读了书。）
3. **дуу дуулгасан** — Бид хүүхдүүдээр дуу **дуулгасан**.（我们让孩子们唱了歌。）
4. **захиа бичүүлсэн** — Тэр надаар захиа **бичүүлсэн**.（他让我写了信。）

### 解析

- 使役式后缀：**-уул- / -лга- / -лгаа-** 等（具体形式因动词而异）
- 结构：使动者 + 被使者（给格/宾格）+ 使役动词
- 常见使役形式：хийх → хийлгэх, унших → уншуулах, дуулах → дуулах, бичих → бичүүлэх

## 三、条件式（-вал / -бал）

用条件式连接两个句子。

### 题目

1. Чи туслана. Би хурдан хийнэ.（你帮忙。我很快就做完。）→ Чи _______, би хурдан хийнэ.
2. Би завтай болно. Би ярина.（我有空。我说。）→ Би _______, би ярина.
3. Тэр ирнэ. Бид хоол идье.（他来。我们吃饭吧。）→ Тэр _______, бид хоол идье.
4. Цас орно. Би гэртээ үлдэнэ.（下雪。我待在家里。）→ Цас _______, би гэртээ үлдэнэ.

### 答案

1. **туславал** — Чи **туславал**, би хурдан хийнэ.（如果你帮忙，我很快就做完。）
2. **завтай болвол** — Би **завтай болвол**, би ярина.（如果我有空，我就说。）
3. **ирвэл** — Тэр **ирвэл**, бид хоол идье.（如果他来，我们就吃饭吧。）
4. **орвол** — Цас **орвол**, би гэртээ үлдэнэ.（如果下雪，我就待在家里。）

### 解析

- 条件式后缀：**-вал / -вэл / -вол / -вөл**（根据元音和谐）
- 结构：条件从句（-вал）+ 主句
- 注意词干变化：туслах → тусла**вал**, ирэх → ир**вэл**, орох → ор**вол**

## 四、副动词（-ж / -ч / -аад / -сан / -магц / -вал 等）

选择正确的副动词形式填空。

### 题目

1. Би ном унш______ унтсан.（我读了书然后睡了。）
2. Тэр ир______ ярья.（等他来了我们再聊。）
3. Би гэртээ харь______ хоол иднэ.（我回家然后吃饭。）
4. Тэр захиа бич______ унтсан.（他写完信就睡了。）

### 答案

1. **ж** — Би ном унш**ж** унтсан.（-ж 表连续动作）
2. **магц** — Тэр ир**мэгц** ярья.（-магц 表"一……就……"）
3. **аад** — Би гэртээ хари**аад** хоол иднэ.（-аад 表完成一个动作后做另一个）
4. **аад** — Тэр захиа бич**ээд** унтсан.（-аад 表完成一个动作后做另一个）

### 解析

- **-ж / -ч**：连接两个同时或连续发生的动作
- **-аад / -ээд / -оод / -өөд**：完成一个动作后做另一个
- **-магц / -мэгц**："一……就……"
- **-вал / -вэл**：条件副动词

## 五、形动词（-х / -даг / -сан / -аад）

选择正确的形动词形式填空。

### 题目

1. Би _______ хүн хүлээж байна.（我在等一个要来的人。）
2. Энэ _______ ном маш сонирхолтой.（这本正在读的书很有趣。）
3. _______ хүн энд байна уу?（有认识他的人吗？）
4. Би өчигдөр _______ кино үзсэн.（我看了昨天说的那部电影。）

### 答案

1. **ирэх** — Би **ирэх** хүн хүлээж байна.（将来时形动词 -х）
2. **уншиж байгаа** — Энэ **уншиж байгаа** ном маш сонирхолтой.（进行时形动词）
3. **Түүнийг мэдэх** — **Түүнийг мэдэх** хүн энд байна уу?（一般形动词 -х）
4. **ярисан** — Би өчигдөр **ярисан** кино үзсэн.（过去时形动词 -сан）

### 解析

- **-х**：将来时形动词（要做……的）
- **-даг / -даг**：习惯/现在时形动词（常做……的）
- **-сан / -сон**：过去时形动词（做过……的）
- **-ж байгаа**：进行时形动词（正在做……的）

## 六、动词的态（主动/被动/使役/互动/共动/众动）

判断下列句子属于哪种"态"，并说明理由。

### 题目

1. Би ном уншсан.（我读了书。）
2. Ном багшаар уншигдсан.（书被老师读了。）
3. Би хүүхдээр хоол хийлгэсэн.（我让孩子做了饭。）
4. Бид хоёулаа дуу дуулсан.（我们俩一起唱了歌。）
5. Монголчууд морь уралддаг.（蒙古人赛马。）

### 答案

1. **主动态** — 主语直接执行动作（би → уншсан）
2. **被动态** — 主语是受动者，施动者用工具格（ном → уншигдсан, багшаар）
3. **使动态** — 主语让别人做动作（би → хийлгэсэн, хүүхдээр）
4. **共动态** — 两人一起做动作（хоёулаа → дуулсан）
5. **众动态** — 多人一起进行竞争性活动（уралддаг）

### 解析

蒙古语动词有六种"态"：
- **主动态** — 主语直接执行动作
- **被动态** — 主语承受动作（-гд-）
- **使动态** — 主语让别人做动作（-уул- / -лга-）
- **互动态** — 双方互相做动作（-лд-）
- **共动态** — 两人一起做动作（-лц-）
- **众动态** — 多人一起做/竞争（-цгаа- / -цгээ-）

## 七、综合对话

将下列中文对话翻译成蒙古语，需使用指定的语法结构。

### 题目

**场景**：你在餐厅遇到朋友，讨论昨晚的经历和周末计划。

A: 你好！昨晚你去哪里了？（用过去时）
B: 我去看了朋友推荐的电影。（用过去时 + 形动词 -сан）
A: 怎么样？好看吗？（用形容词 + 疑问句）
B: 很好看。如果你周末有空，我们一起去看吧！（用条件式 + 建议）
A: 好啊。但是如果下雨怎么办？（用条件式）
B: 如果下雨，我们就待在家里做饭。我让孩子们帮忙。（用被动/使役）
A: 太好了！那周末见！

### 答案

A: Сайн байна уу! Өчигдөр орой чи хаашаа явсан бэ?

B: Би найзын маань зөвлөсөн кино үзэхээр явсан.

A: Ямар байсан бэ? Сайн уу?

B: Их сайн. Хэрвээ чи амралтын өдөр завтай болвол, бид хамт явж үзье!

A: Сайн шүү. Гэвч хэрвээ бороо орвол яах вэ?

B: Хэрвээ бороо орвол, бид гэртээ хоол хийе. Би хүүхдүүдээр туслуулна.

A: Гайхалтай! Тэгвэл амралтын өдөр уулзъя!

### 解析

- "昨晚" — өчигдөр орой
- "朋友推荐的" — найзын маань зөвлөсөн（过去时形动词 -сан）
- "如果……" — хэрвээ ... -вал/-вэл/-вол/-вөл（条件式）
- "一起去看" — хамт явж үзье（-ж + 祈使式）
- "让孩子们帮忙" — хүүхдүүдээр туслуулна（使役式 -уул-）
