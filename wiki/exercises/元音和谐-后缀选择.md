# 元音和谐：后缀选择练习

## 目标

练习判断词根元音类型，并选择合适的后缀元音。

## A. 判断元音类型

| 单词 | 元音类型 |
|------|----------|
| ном | ____ |
| хүн | ____ |
| багш | ____ |
| гэр | ____ |
| анги | ____ |

可选：后元音 / 前元音 / 只有中性元音

## B. 选择属格

选择 `-ын` 或 `-ийн`：

1. ном + ____
2. гэр + ____
3. анги + ____
4. багш + ____

## C. 选择离格

选择 `-аас/-ээс/-оос/-өөс`：

1. Америк + ____
2. хот + ____
3. гэр + ____
4. Хөвсгөл + ____

## 答案

### A

| 单词 | 元音类型 |
|------|----------|
| ном | 后元音 |
| хүн | 前元音 |
| багш | 后元音 |
| гэр | 前元音 |
| анги | 只有中性元音，通常按前元音处理 |

### B

1. ном + -ын → номын
2. гэр + -ийн → гэрийн
3. анги + -ийн → ангийн
4. багш + -ийн → багшийн

### C

1. Америкаас
2. хотоос
3. гэрээс
4. Хөвсгөлөөс

## 备注

本页保留 `багшийн` 这类容易误判的形式，提醒不要把元音和谐律简化成唯一规则。后缀选择还会受词尾音、拼写习惯和具体词形影响。
