# 阶段5：交通与旅行练习

> 本练习覆盖 Peace Corps Unit 4 (Traveling) 的核心语法点：方向词、位置后置词、向格 руу/луу、工具格（交通方式）、请求许可 -ж болох уу、礼貌命令 -аарай4、反身领属 -аа4。

---

## 一、填空：向格 -руу / -луу

用正确的向格形式填空。

### 题目

1. Би Дархан _______ явмаар байна.
2. Би Хонгор _______ явна.
3. Эндээс Ховд _______ юугаар явдаг вэ?
4. Чи Хэнтий _______ яаж явах вэ?
5. Би дэлгүүр _______ таксигаар явмаар байна.

### 答案

1. **руу** — Би Дархан **руу** явмаар байна.（我想去达尔汗。）
2. **луу** — Би Хонгор **луу** явна.（我去洪戈尔。）
3. **руу** — Эндээс Ховд **руу** юугаар явдаг вэ?（从这里怎么去霍夫德？）
4. **руу** — Чи Хэнтий **руу** яаж явах вэ?（你怎么去肯特？）
5. **лүү** — Би дэлгүүр **лүү** таксигаар явмаар байна.（我想坐出租车去商店。）

### 解析

- 大多数名词接 **-руу / -рүү**
- 以 -р 结尾的名词接 **-луу / -лүү**（避免双写 r）
- 元音和谐：后元音接 -руу，前元音接 -рүү

## 二、选择：位置后置词

选择正确的位置表达（дээр / доор / дотор / хажууд / урд / ард / хооронд / гадаа）。

### 题目

1. Ширээ______ ном байна.（书在桌子上。）
2. Ширээ______ хүүхэд байна.（孩子在桌子下面。）
3. Баг______ харандаа байна.（铅笔在包里。）
4. Сургуулийн______ цэцэрлэг байна.（幼儿园在学校前面。）
5. Театрын______ байна.（在剧院外面。）

### 答案

1. **н дээр** — Ширээ**н дээр** ном байна.（дээр 前只接 -н）
2. **н доор** — Ширээ**н доор** хүүхэд байна.（доор 前只接 -н）
3. **ний дотор** — Баг**ний дотор** харандаа байна.（дотор 前只接 -н）
4. **урд** — Сургуулийн **урд** цэцэрлэг байна.（其他后置词前接完整属格 -ийн）
5. **гадна** — Театрын **гадна** байна.（其他后置词前接完整属格 -ын）

### 解析

- **дээр / доор / дотор** 前面只接 **-н**（不接完整属格）
- **хажууд / урд / өмнө / ард / цаана / хооронд / гадаа / гадна** 前面接**完整属格**（-ийн/-ын/-ний/-н）

## 三、翻译：问路对话

将下列中文翻译成蒙古语。

### 题目

1. 通讯中心在哪里？
2. 在食品市场的右边。
3. 怎么去那里？
4. 先直走，然后左转。
5. 请右转。

### 答案

1. **Холбоо хаана байдаг юм бэ?**
2. **Хүнсний захын баруун талд.**
3. **Тийшээ яаж очих вэ?**
4. **Эхлээд чигээрээ яваад, тэгээд зүүн тийшээ эргэ.**
5. **Баруун тийшээ эргээрэй.**

### 解析

- "在哪里"用 **хаана байдаг вэ?**
- "在……右边"用 **...-ын баруун талд**
- "怎么去"用 **яаж очих вэ?**
- "先……然后……"用 **эхлээд... тэгээд...**
- "请……"用动词 + **-аарай4**（礼貌命令式）

## 四、选择：工具格（交通方式）

选择正确的交通工具形式填空。

### 题目

1. Би сургуульдаа автобус_______ явдаг.
2. Би Кэмбрижид галт тэрэг_______ ирсэн.
3. Та нар Дархан руу юугаар явсан бэ? — _______（坐公共汽车）
4. Чи Хэнтий рүү яаж явах вэ? — Машин_______ эсвэл онгоц_______.
5. Би мор_______ ирсэн.

### 答案

1. **аар** — автобус**аар**
2. **ээр** — галт тэрэг**ээр**
3. **Автобусаар** — Автобус**аар**
4. **аар** / **оор** — машин**аар** эсвэл онгоц**оор**
5. **иор** — мор**иор**

### 解析

- 工具格后缀：**-аар / -ээр / -оор / -өөр**
- 隐藏 g 名词/长元音结尾加 **-г**：морь → мор**иор**
- 询问方式用 **юугаар?** 或 **яаж?**

## 五、填空：反身领属 -аа4

用正确的反身领属形式填空。

### 题目

1. Би гэр_______ явна.（我回自己的家。）
2. Чи билет_______ авсан уу?（你买你的票了吗？）
3. Би алим_______ идсэн.（我吃了我的苹果。）
4. Тэр гэр_______ гарсан.（他从自己的家里出来了。）
5. Би гэр_______ очиж амарсан.（我回自己的家休息了。）

### 答案

1. **лүүгээ** — Би гэр **лүүгээ** явна.（向格 + 反身领属，长元音后加 г）
2. **ээ** — Чи билет**ээ** авсан уу?
3. **аа** — Би алим**аа** идсэн.
4. **ээсээ** — Тэр гэр**ээсээ** гарсан.（离格 + 反身领属）
5. **тээ** — Би гэр**тээ** очиж амарсан.（位格 + 反身领属）

### 解析

- 反身领属后缀：**-аа / -ээ / -оо / -өө**
- 与向格 -луу/-руu 连用时，长元音/双元音后加 **г**：гэр **лүүгээ**
- 反身领属可以加在各格形式之后

## 六、填空：请求许可（-ж / -ч болох уу?）

用正确的 -ж 或 -ч 形式填空。

### 题目

1. Энд су_______ болох уу?（我可以坐这里吗？）
2. ... ав_______ болох уу?（我可以拿……吗？）
3. ... хар_______ болох уу?（我可以看……吗？）
4. ... үз_______ болох уу?（我可以看看……吗？）
5. ... өг_______ болох уу?（我可以给……吗？）

### 答案

1. **ж** — Энд су**ж** болох уу?
2. **ч** — ав**ч**（авах → авч）
3. **ж** — хар**ж**（харах → харж）
4. **ж** — үз**ж**（үзэх → үзэж）
5. **ч** — өг**ч**（өгөх → өгч）

### 解析

- 大多数动词接 **-ж**：суух → суж, харах → харж, үзэх → үзэж
- 以 -р, -в, -г 结尾的动词接 **-ч**：авах → авч, өгөх → өгч, сурах → сурч
- 结构：动词词干 + -ж/-ч + болох уу?

## 七、改错

找出下列句子中的错误并改正。

### 题目

1. Би Дархан явмаар байна.
2. Ширээний дээр ном байна.
3. Би гэр лүүээ явна.
4. Би автобус явсан.
5. Сургуульд урд цэцэрлэг байна.

### 答案

1. **Дархан** → **Дархан руу** явмаар байна.
   - 错误：缺少向格 -руу

2. **Ширээний дээр** → **Ширээн дээр** ном байна.
   - 错误：дээр 前只用 -н，不用完整属格

3. **гэр лүүээ** → **гэр лүүгээ** явна.
   - 错误：向格 + 反身领属时，长元音后需加 г

4. **автобус** → **автобусаар** явсан.
   - 错误：缺少工具格 -аар

5. **Сургуульд урд** → **Сургуулийн урд** цэцэрлэг байна.
   - 错误：урд 前需用完整属格 -ийн，不是位格 -д
