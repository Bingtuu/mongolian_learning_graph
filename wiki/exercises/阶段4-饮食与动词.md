# 阶段4：饮食与动词练习

> 本练习覆盖 Peace Corps Unit 3 (Food) 的核心语法点：食物词汇、дуртай/дургүй、愿望式 -маар байна、动词作形容词 -сан、量词用法。

---

## 一、填空：дуртай / дургүй

用正确的形式填空（注意位格 -нд 的使用）。

### 题目

1. Би алим______ дуртай.
2. Тэр загас______ дургүй.
3. Чи ямар жимс______ дуртай вэ?
4. Би тарганд, сүү______ дуртай.
5. Монголчууд мах______ дуртай.

### 答案

1. **анд** — Би алим**анд** дуртай.（我喜欢苹果。）
2. **анд** — Тэр загас**анд** дургүй.（他不喜欢鱼。）
3. **энд** — Чи ямар жимс**энд** дуртай вэ?（你喜欢哪种水果？）
4. **нд** — Би тарганд, сүү**нд** дуртай.（我喜欢酸奶和牛奶。）
5. **анд** — Монголчууд мах**анд** дуртай.（蒙古人喜欢肉。）

### 解析

- дуртай/дургүй 的对象必须用**位格/给格 -нд**
- 后元音词（а, о, у）接 **-анд/онд/унд**
- 前元音词（э, ө, ү）接 **-энд/өнд/үнд**
- 中性元音 и 后的规则参见 [[元音和谐律]]

## 二、改写：愿望式 -маар байна

将下列句子改写为"想做……"的形式。

### 题目

1. Би өндөг идсэн. → Би өндөг ид______ байна.
2. Би монгол хэл сурна. → Би монгол хэл сур______ байна.
3. Бид нар хөл бөмбөг тоглосон. → Бид нар хөл бөмбөг тогл______ байна.
4. Би нэг юм иднэ. → Би нэг юм ид______ байна.
5. Би маргааш Дархан явна. → Би маргааш Дархан яв______ байна.

### 答案

1. **мээр** — Би өндөг ид**мээр** байна.（我想吃鸡蛋。）
2. **маар** — Би монгол хэл сур**маар** байна.（我想学蒙古语。）
3. **моор** — Бид нар хөл бөмбөг тогл**моор** байна.（我们想踢足球。）
4. **мээр** — Би нэг юм ид**мээр** байна.（我想吃点东西。）
5. **маар** — Би маргааш Дархан яв**маар** байна.（我想明天去达尔汗。）

### 解析

- 愿望式后缀 **-маар / -мээр / -моор / -мөөр** 根据元音和谐选择
- 结构：动词词干 + -маар + байна（肯定）/ байна.гүй（否定）
- 否定形式：-маар**гүй** байна

## 三、选择：动词作形容词

选择合适的 -сан / -сан"гүй" 形式填空。

### 题目

1. Би ______ өндөг идсэн.（我吃了煎鸡蛋。）
   a. шарсан  b. шараагүй
2. Би ______ өндөг иддэггүй.（我不吃没煎的鸡蛋。）
   a. шарсан  b. шараагүй
3. ______ будаа маш амттай.（蒸米饭很好吃。）
   a. Жигнэсэн  b. Жигнээгүй
4. Буцлаагүй ус уух ______.（不喝生水。）
   a. сайн  b. муу
5. Дорж ______ ус уув.（道尔吉喝了开水。）
   a. буцалсан  b. буцлаагүй

### 答案

1. **a** — Би **шарсан** өндөг идсэн.
2. **b** — Би **шараагүй** өндөг иддэггүй.
3. **a** — **Жигнэсэн** будаа маш амттай.
4. **b** — Буцлаагүй ус уух **муу**.
5. **a** — Дорж **буцалсан** ус уув.

### 解析

- 动词过去时 -сан 放在名词前可以作形容词，表示"已……的"
- 否定形式 -аагүй（或 -ээгүй/-оогүй/-өөгүй）表示"未……的"
- 这是蒙古语中非常常见的修饰结构

## 四、填空：量词与数量词

选择合适的量词填空（лааз, шил, уут, аяга, кило, ширхэг, литр）。

### 题目

1. Нэг ______ пиво.
2. Хоёр ______ сүү.
3. Гурван ______ гурил.
4. Нэг ______ цай.
5. Таван ______ мах.
6. Долоон ______ чихэр.

### 答案

1. **лааз** — Нэг **лааз** пиво.（一罐啤酒。）
2. **литр** — Хоёр **литр** сүү.（两升牛奶。）
3. **уут** — Гурван **уут** гурил.（三袋面粉。）
4. **аяга** — Нэг **аяга** цай.（一碗茶。）
5. **кило** — Таван **кило** мах.（五公斤肉。）
6. **ширхэг** — Долоон **ширхэг** чихэр.（七颗糖果。）

### 解析

- 蒙古语量词位于数词和名词之间：**数词 + 量词 + 名词**
- лааз（罐）— 罐装饮料
- шил（瓶）— 瓶装液体
- уут（袋）— 袋装物品
- аяга（碗）— 碗装食物/饮料
- кило（公斤）— 重量
- ширхэг（个）— 个体计数
- литр（升）— 液体体积

## 五、翻译：餐厅对话

将下列中文翻译成蒙古语。

### 题目

1. 我想吃五个包子。
2. 有什么菜？
3. 包子卖完了！
4. 我想结账。
5. 我饱了。够了。

### 答案

1. **Таван бууз идмээр байна.**
2. **Ямар хоол гарсан бэ?**
3. **Бууз дууссан шүү!**
4. **Тооцоо хиймээр байна.**
5. **Би цадсан. Болно, болно.**

### 解析

- "想做"用 **-маар байна**
- "有什么"用 **... байна уу?** 或 **ямар ...?**
- "卖完了"用 **дууссан**（过去时）
- "结账"是 **тооцоо хийх**
- "饱了"是 **цадсан**（过去时）
- "够了"是 **Болно!**

## 六、改错

找出下列句子中的错误并改正。

### 题目

1. Монголчууд мах дуртай.
2. Та сүүнд дуртай вэ?
3. Чи ямар хоол дургүй вэ?
4. Би алиманд, банан, жүржэнд дуртай.

### 答案

1. **Монголчууд мах** → **Монголчууд маханд** дуртай.
   - 错误：дуртай 的对象缺少位格 -нд

2. **Та сүүнд дуртай вэ?** → **Та сүүнд дуртай юу?**
   - 错误：一般疑问句应该用 юу，不是 вэ

3. **Чи ямар хоол дургүй вэ?** → **Чи ямар хоолонд дургүй вэ?**
   - 错误：дургүй 的对象缺少位格 -нд

4. **Би алиманд, банан, жүржэнд дуртай.** → **Би алиманд, банан**д**, жүржэнд дуртай.**
   - 错误：банан 缺少位格 -д

### 解析

- дуртай/дургүй 的**每一个对象**都必须接位格 -нд
- 一般疑问句用 **юу**，特殊疑问句用 **вэ**
- 列举时每个名词都要变格
