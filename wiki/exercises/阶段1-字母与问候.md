# 阶段1：字母与问候练习

## 目标

完成本页后，你应该能读出基础问候句，区分正式/非正式“你”，并能选择 `уу/үү/юу/юү` 这类一般疑问句语气词。

## A. 朗读

朗读并写出中文意思：

1. Та сайн байна уу?
2. Сайн уу?
3. Сонин юу байна?
4. Юмгүй дээ.
5. Баяртай!

## B. 选择：та 还是 чи

为下面场景选择更合适的“你”：

| 场景 | 选择 |
|------|------|
| 第一次见老师 | ____ |
| 和同龄朋友说话 | ____ |
| 对长辈说话 | ____ |
| 和熟悉的弟弟妹妹说话 | ____ |

## C. 填空

用 `уу/үү/юу/юү` 填空：

1. Сайн байна ____?
2. Сайн ____?
3. Байгаа ____?

## 答案

### A

1. 您好/您好吗？
2. 你好？
3. 有什么新鲜事？
4. 没什么。
5. 再见。

### B

| 场景 | 选择 |
|------|------|
| 第一次见老师 | та |
| 和同龄朋友说话 | чи |
| 对长辈说话 | та |
| 和熟悉的弟弟妹妹说话 | чи |

### C

1. Сайн байна уу?
2. Сайн уу?
3. Байгаа юу?
