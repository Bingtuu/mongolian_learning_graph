# 家庭与人物 (Family and People / гэр бүл, хүмүүс)

## 家庭成员词汇

| 蒙古语 | 拉丁转写 | 中文 | 备注 |
|--------|----------|------|------|
| аав | aav | 父亲 | |
| ээж | eej | 母亲 | |
| ах | akh | 哥哥 | 也用于称呼年长于己的男性 |
| эгч | egch | 姐姐 | 也用于称呼年长于己的女性 |
| дүү | düü | 弟弟/妹妹 | 不分性别，需进一步询问 |
| эмээ | emee | 祖母 | |
| өвөө | övöö | 祖父 | |
| эхнэр | ekhner | 妻子 | |
| нөхөр | nökhör | 丈夫 | |
| хүүхэд | khüükhéd | 孩子 | |
| хүргэн | khürgen | 女婿 | |
| бэр | ber | 儿媳 | |
| найз | naiz | 朋友 | |
| гэр бүл | ger bül | 家庭 | |
| гэрийнхэн | geriinkhen | 家人 | 属格 + хэн 的集合形式 |

> **文化提示**：蒙古语中 `ах`、`эгч`、`дүү` 的使用比英语 brother/sister 更广泛。年长的男性/女性熟人、亲戚、甚至陌生人都可以用 `ах` / `эгч` 称呼；比自己年幼的人可用 `дүү`。
> — Peace Corps Unit 2 Lesson 1 p.39 Cultural Note

## 核心句型拆解

### 介绍家人

> Энэ манай гэрийнхэн.
> Ene manai geriinkhen.
> энэ / манай / гэрийнхэн
> 这 / 我们的 / 家人
> (这是我的家人。)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[名词谓语]]、[[属格的形成]]

> Энэ хэн бэ?
> Ene khen be?
> энэ / хэн / бэ
> 这 / 谁 / (疑问)
> (这是谁？)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[疑问句]]

> Энэ манай ээж.
> Ene manai eej.
> энэ / манай / ээж
> 这 / 我的 / 母亲
> (这是我母亲。)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[名词谓语]]

> Танай ээжийг хэн гэдэг вэ?
> Tanai eejig khen gedeg ve?
> танай / ээжийг / хэн / гэдэг / вэ
> 你的 / 母亲(宾格) / 谁 / 叫做 / (疑问)
> (你母亲叫什么名字？)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[名词变格]]、[[疑问句]]

> Манай ээжийг Дулам гэдэг.
> Manai eejig Dulam gedeg.
> манай / ээжийг / Дулам / гэдэг
> 我的 / 母亲(宾格) / 杜拉姆 / 叫做
> (我母亲叫杜拉姆。)

> Энэ чиний дүү мөн үү?
> Ene chinii düü mön üü?
> энэ / чиний / дүү / мөн / үү
> 这 / 你的 / 弟/妹 / 是 / 吗
> (这是你的弟弟/妹妹吗？)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[名词谓语]]、[[疑问句]]

> Биш. Манай ах.
> Bish. Manai akh.
> биш / манай / ах
> 不是 / 我的 / 哥哥
> (不是。是我哥哥。)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[否定式]]、[[名词谓语]]

> Харин энэ хүн хэн бэ?
> Kharin ene khün khen be?
> харин / энэ / хүн / хэн / бэ
> 那/那么 / 这 / 人 / 谁 / (疑问)
> (那这个人是谁？)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[疑问句]]

> Манай ахын эхнэр.
> Manai akhyn ekhner.
> манай / ахын / эхнэр
> 我的 / 哥哥的 / 妻子
> (是我哥哥的妻子。)
> — 来源: Peace Corps Unit 2 Lesson 1 p.34 | 语法点: [[属格的形成]]、[[名词谓语]]

### 年龄与婚姻

> Ах чинь хэдэн настай вэ?

## 人物状态

> Баатарын бие сайн, харин дүү нь өвчтэй гэнэ.
> Baataryn bie sain, kharin düü növchtee gene.

> 巴特尔的 / 身体 / 好 / 但是 / 弟弟/妹妹 / 他的 / 生病 / 据说
> (巴特尔身体很好，但据说他弟弟生病了。)
> — 来源: [[Gaunt 现代蒙古语]] 第5章 | 语法点: [[属格的形成]]、[[主语标记 нь]]

> Ээжийн чинь санаа одоо их зовж байна.

> 母亲的 / 你的 / 心思 / 现在 / 很 / 担忧 / 正在

> — 来源: [[Gaunt 现代蒙古语]] 第6章 | 语法点: [[属格的形成]]、[[主语标记 нь]]
